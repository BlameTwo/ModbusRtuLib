#### Qna3E-SerialPort

说明：三菱Qnc3E 帧结构报文协议

物理介质：串口

| 序号    | 说明         | 示例        | 占用字节 | 备注                                                       |
| ----- | ---------- | --------- | ---- | -------------------------------------------------------- |
| 1,2   | 固定头        | 0x50 0x00 | 2    |                                                          |
| 3     | 网络号        | 0x00      | 1    |                                                          |
| 4     | 网络地址（广播地址） | 0xFF      | 1    |                                                          |
| 5,6   | CPU目标编号    | 0xFF 0x03 | 2    |                                                          |
| 7     | 网络站号       | 0x00      | 1    |                                                          |
| 8,9   | PLC回复延迟    | 0x0A 0x0A | 2    | 单位计数，例如1个单位就是10毫秒，short高低位置换 |
| 10,11 | 主命令        |           | 2    |                                                          |
| 12,13 | 子命令        |           | 2    |                                                          |
